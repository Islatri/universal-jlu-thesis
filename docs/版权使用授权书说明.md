# 版权使用授权书页面 - 使用说明

## 功能概述

已在模板中新增"版权使用授权书"页面，该页面会自动插入到承诺书（commitment page）之后。

## 页面参数

在 `jlu-bachelor.with()` 调用中添加以下参数来自定义授权书：

### 参数列表

| 参数名                       | 类型               | 默认值  | 说明                                                  |
| ---------------------------- | ------------------ | ------- | ----------------------------------------------------- |
| `authorization-confidential` | `bool`             | `false` | 是否保密。设为 `true` 为保密，`false` 为不保密        |
| `authorization-author-sig`   | `str \| none`      | `none`  | 作者签名图片的路径。如果为 `none`，显示空白签名线     |
| `authorization-mentor-sig`   | `str \| none`      | `none`  | 指导教师签名图片的路径。如果为 `none`，显示空白签名线 |
| `authorization-date`         | `datetime \| auto` | `auto`  | 签署日期。`auto` 表示使用当前日期，也可指定具体日期   |

## 使用示例

### 基础用法（使用所有默认值）

```typst
#show: jlu-bachelor.with(
  // ... 其他参数 ...
  // 不指定授权书参数，将使用默认配置
)
```

### 指定签名图片

```typst
#show: jlu-bachelor.with(
  // ... 其他参数 ...
  
  authorization-author-sig: "assets/images/author-signature.png",
  authorization-mentor-sig: "assets/images/mentor-signature.png",
)
```

### 设置保密和日期

```typst
#show: jlu-bachelor.with(
  // ... 其他参数 ...
  
  authorization-confidential: true,  // 设为保密
  authorization-date: datetime(year: 2024, month: 10, day: 25),
)
```

### 完整配置示例

```typst
#show: jlu-bachelor.with(
  // ... 其他参数（thesis-info, abstract等）...
  
  // 版权使用授权书配置
  authorization-confidential: false,
  authorization-author-sig: "assets/images/author-signature.png",
  authorization-mentor-sig: "assets/images/mentor-signature.png",
  authorization-date: datetime(year: 2024, month: 10, day: 25),
)
```

## 签名图片准备

### 推荐规格

- **格式**：PNG 或 JPG
- **尺寸**：建议 800×400 像素或更大以保证清晰度
- **背景**：透明背景（PNG）或白色背景
- **内容**：仅包含清晰的手写签名

### 存放位置

将签名图片放在论文项目的 `assets/images/` 目录下，然后在参数中引用相对路径：

```
project/
├── jlu-bachelor-thesis.typ
├── assets/
│   └── images/
│       ├── author-signature.png
│       └── mentor-signature.png
└── ...
```

## 默认内容

授权书的文本内容已预设为符合吉林大学要求的标准版本。如需修改版权声明文本，可以：

1. 编辑 `core/authorization.typ` 文件中的 `content` 参数
2. 在调用 `authorization-page()` 时传入自定义内容

## 保密期限

如果选择保密（`authorization-confidential: true`），授权书上会显示保密期限输入框。指导教师应在该处填写具体的解密年份。

## 技术细节

- 授权书实现在 `core/authorization.typ` 文件中
- 在主模板 `core/template.typ` 中自动调用该页面
- 页面编号延续论文总体编号，不单独编号
