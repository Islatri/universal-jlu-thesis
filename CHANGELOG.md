# Changelog

## 0.1.7

- 初始版本发布

## 0.1.8

- 新增版权使用授权书页面，自动插入到承诺书之后
- 支持以下授权书参数配置：
  - `authorization-confidential`：设置是否保密
  - `authorization-author-sig`：上传作者签名图片
  - `authorization-mentor-sig`：上传指导教师签名图片
  - `authorization-date`：自定义签署日期
- 新增 `core/authorization.typ` 模块实现授权书页面
- 新增《版权使用授权书说明.md》使用文档
